The Midnight Project

Data entry application targeted at Multiple Sclerosis patients

The Midnight Project is program used to help Multiple Sclerosis patients keep track of everyday symptoms.
It consists of a questionnaire for patients to enter data and a database that stores the patient's day to day data and notifies the patient via email if the analyzed data indicates a negative trend.
The questionnaire was built using QT4 and Python. The database was created with MySQL, Microsoft Access and Microsoft Excel.

Created by Kevin Huo, Toni Ogunmade, Herman Grewal and Mohamed El Shatshat for Hack 4 Health 2015. It was selected as one the top ten hacks at the event.
