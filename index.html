The Learner

A robot created to learn SLAM and carlike vehicle controls

The learner is an Arduino powered mobile robot that was built to perform passive SLAM. It is controlled over bluetooth with a PS3 controller connected to a computer running ROS. The robot has 7 ultrasonic sensors, an IMU and a GPS unit and filters the sensor data on its microcontroller. The filtered data is then sent to a the PC to create a map of the robots surroundings.


The electrical distribution board was designed in EagleCAD and soldered by hand. The sensors utilize UART and I2C protocol to communicate and have been calibrated for this specific purpose.
The visualization of the data is done using rViz

To perform SLAM, the code is designed to assume known poses generated from a multisensor Extended Kalman Filter outlined here. The ultrasonic sensors are then used to create an occupancy grid that would describe the area being mapped. The Kalman filter has been design and is currently being implemented the mapping will follow soon after.

GPS I2C

An Embedded unit to provide GPS data over I2C

It is comprises of an Atmel Atmega328 microcontroller and a GPS module. The microcontroller parses NMEA sentences generated by the GPS almost instantaneously using it's USART interrupts. It provides the data over I2C per request in the form of a character array with a one byte checksum to help validate the data.

Maestro

Leap motion ROS integration


Maestro provides an individual with a means of controlling different robots using hand gestures. It is currently setup to control a Clearpath Husky but can be configured to control ground and aerial robots.
Won first place at the Deloitte Tech Exchange.
The code was written in Python using the Leap motion API and ROS

Created by Kevin Huo, Herman Grewal, Hamdan Chaudry, Neil Wadhvana, Sukhraj Hothi, Rayyan Ghani and Oluwatoni Ogunmade for Deloitte Tech Exchange 2015.

FaceTrack

Face tracking gimbal

FaceTrack consists of a camera attached to a two axis gimbal. It tracks faces using a Haar cascade loaded unto OpenCV and uses the gimbal to keep the tracked face within view. It uses PID controllers to control the gimbal.

Created by Kevin Huo and Oluwatoni Ogunmade

The Midnight Project

Data entry PC application targeted at Multiple sclerosis patients

The Midnight Project is a PC program to help Multiple Sclerosis patients keep track of everyday symptoms.
It consists of a questionnaire for patients to enter data and a database that stores the patient's day to day data and notifies the patient via email if the analyzed data indicates a negative trend.
The questionnaire was built using QT4 and Python. The database was created with SQL, Microsoft Access and Microsoft Excel.

Created by Kevin Huo, Mohammed El-Shatshat, Herman Grewal and Toni Ogunmade for Hack 4 Health 2015. It was selected as one the top ten hacks at the event

Psycle

Bicycle accessory for easy navigation

A system to make your bicycle smarter, safer, and more secure. The Psycle brings your bicycle into the 21st century by combining an Android application for smartphones with a multitude of sensors powered by an Arduino and the Intel Edison board. It provides turn-by-turn navigation using Google Maps without the need to look at a phone through vibrating motors on each handlebar and also controls turn signalling via Bluetooth communication. A basic anti-theft system is in place as well as an LED screen which displays vital information for cyclists.
The Arduino and Intel Edison code was written in C++ and the Android app was written in Java using the Android Studio IDE

Created by Archie Lee, Jerry Li, Oluwatoni Ogunmade for WearHacks Toronto 2015.

ScanTRON 2019

Multiple choice marking robot

The scanTRON 2019 is a collaborative lego mindstorms project aimed at solving the tedious task of marking multiple choice answer sheets by hand. The scanTron is faster than a human marker and because of that it gives professors more time to attend to other tasks. Furthermore, it provides a representation of student’s scores in a histogram format.
The code was written entirely in RobotC.

Created by Ben Milligan, Ilia Poretski, Johnathan Van Gorp and Oluwatoni Ogunmade for MTE 100

Absolute Linear Encoder

An accurate way to measure displacement

A measurement device developed the for an open ended statistics design project. It measures displacements up to 15cm with 0.625mm accuracy.
